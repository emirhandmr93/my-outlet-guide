MasterData FINAL v1

This folder is the single source of truth for app content generation.

CSV conventions:
- Multi-value fields use semicolon (;), for example services or restaurant lists.
- Ordered step lists use pipe (|), for example 1:Step one|2:Step two.
- Boolean values use TRUE/FALSE.
- status should usually be active/inactive.
